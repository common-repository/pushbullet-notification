Pushbullet Notifications allows your WordPress site to send push notifications straight to your Android and iOS device.

== Description ==

Using the Pushbullet Notifications application on your Android device in conjunction with your WordPress blog and this plugin, you can be notified of events happening on your blog as Push Notifications on your mobile device.

Currently supports notifications for new users, comments, pingback/trackbacks, user specific password reset notifications, plugin and theme upgrades, core upgrades.

== Installation ==

You can install this plugin directly from your WordPress dashboard:

 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *Pushbullet notifications*.
 3. Click *Install Now* next to the *Pushbullet notifications* plugin.
 4. Activate the plugin.

Alternatively, see the guide to [Manually Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Usage =

1. Go to the Settings -> Pushbullet Notifications.
